HTML5, CSS3, JavaScript 한눈에 보기

이 예제는 한 파일 안에 웹의 기본 구조와 스타일, 그리고 간단한 동작까지 모두 담은 학습용 페이지입니다. 오른쪽 버튼으로 다크 모드를 바꾸고, 아래 예제로 HTML5 시맨틱 태그, CSS3 스타일, JavaScript 이벤트를 직접 확인해 보세요.

HTML5 구조 CSS3 디자인 JavaScript 상호작용

웹페이지의 3가지 핵심 요소를 함께 학습

HTML은 뼈대, CSS는 보이는 모습, JavaScript는 반응하는 동작을 담당합니다. 이 페이지는 세 가지를 분리해서 보이면서도 하나의 문서 안에서 함께 움직이도록 구성했습니다.

아래의 예시들은 단순한 설명이 아니라 실제로 페이지에 적용되는 코드 조각입니다. 학습하면서 코드를 바꿔보면 결과가 바로 눈에 보입니다.

HTML5 예시

시맨틱 태그를 사용하면 문서의 의미가 더 명확해집니다.

<header>사이트 제목과 메뉴</header>
<main>
  <section>주요 콘텐츠</section>
  <article>독립적인 글</article>
</main>
<footer>하단 정보</footer>

CSS3 예시

그라디언트, 그림자, 반응형 레이아웃을 함께 사용해 보세요.

.card {
  background: rgba(255,255,255,0.8);
  border-radius: 24px;
  box-shadow: 0 18px 40px rgba(0,0,0,0.12);
}

@media (max-width: 900px) {
  .grid { grid-template-columns: 1fr; }
}

JavaScript 예시

버튼 클릭과 시계 업데이트처럼 간단한 동작을 만들 수 있습니다.

현재 값: 0 
document.querySelector('#addCount').addEventListener('click', () => {
  count += 1;
  countView.textContent = count;
});

학습 포인트

  • HTML5: section, article, aside 같은 의미 있는 구조를 사용합니다.
  • CSS3: 변수, 그라디언트, 반응형 레이아웃, 애니메이션을 확인할 수 있습니다.
  • JavaScript: 버튼 클릭, DOM 선택, 실시간 시계 갱신을 연습할 수 있습니다.
